Popular Algorithms

First-Come, First-Served (FCFS)

FCFS is a non-preemptive scheduling algorithm that serves requests in the order they arrive.

Shortest Job First (SJF)

A non-preemptive scheduling algorithm that selects the process with the smallest burst time next.

Priority Scheduling

Priority Scheduling assigns the CPU to processes based on priority; higher priorities are served first.

Round Robin

Round Robin is a preemptive scheduling algorithm that assigns time slices to each process in a cyclic order.

Learn More Simulate

Selection Sort

Selection Sort divides the input list into two parts: sorted and unsorted, and repeatedly selects the smallest element.

Learn More Simulate

Bubble Sort

Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Learn More Simulate

Insertion Sort

Insertion Sort builds a sorted array one element at a time, inserting elements into their correct position.

Linear Search

Linear Search scans each element in a list to find a target value, returning the index if found.

Binary Search

Binary Search finds the position of a target value within a sorted array by repeatedly dividing the search interval in half.
